What is an HTML iFrame ?

An iframe (short for inline frame) is an HTML element that allows an external webpage to be embedded in an HTML document.

An iframe can be inserted into an HTML document using the <iframe> tag.

iFrames are mainly used for displaying different types of media within a webpage.

Web designers use iFrames to embed interactive applications in Web pages, including YouTube videos, Google Maps or ecommerce applications.

HTML iFrame Attributes

Following are the attributes of an HTML iFrame code:

  • src: Specifies the address of the document to embed in the <iframe>
  • name: Specifies the name of an <iframe>
  • width: Specifies the width of an <iframe>
  • height: Specifies the height of an <iframe>
  • align: Specifies the alignment of an <iframe> according to surrounding elements
  • scrolling: Specifies whether or not to display scrollbars in an <iframe>
